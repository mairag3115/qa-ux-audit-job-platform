\# Bug Report 01



\## Title

Search results page remains dimmed after returning from a job application flow



\## ID

BUG-01



\## Severity

Medium



\## Priority

High



\## Environment

Web browser on desktop



\## Preconditions

The user is logged in and browsing job search results on Computrabajo.



\## Steps to Reproduce

1\. Search for a job position

2\. Open a job offer or start the application flow

3\. Click the browser back button to return to the search results page

4\. Observe the screen state



\## Expected Result

The user should return to the search results page in a normal and fully interactive state.



\## Actual Result

The page appears dimmed or covered by a gray overlay after returning, which makes the interface look blocked or not fully restored.



\## Impact

This may confuse users, affect navigation, and reduce confidence in continuing the job search process.
