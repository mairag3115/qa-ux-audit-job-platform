\# Project Overview



\## Platform Analyzed

Computrabajo



\## Analysis Type

Manual QA + UX Audit



\## Main Goal

To review the platform's main functionality and user experience during the job application process.



\## Target User

A person searching for jobs online, browsing offers, filtering results, and applying to opportunities.



\## Core User Flow

1\. Access the platform

2\. Search for a job

3\. Apply filters

4\. Open a job offer

5\. Review details

6\. Submit an application



\## Areas Evaluated

\- Navigation

\- Search functionality

\- Filter usability

\- Clarity of job listings

\- Application process

\- Error handling

\- Overall usability



\## Expected Outcome

Identify bugs, usability issues, friction points, and improvement opportunities.
