# QA + UX Audit – Job Platform

## Project Overview
This project is a manual QA and UX audit of a job platform. The goal is to evaluate the main user flow, identify functional bugs, detect usability issues, and propose improvements to enhance the user experience.

## Objective
To simulate a real QA and UX review process by analyzing a job search platform from both a testing and usability perspective.

## Scope
The project evaluates the following user flow:
- User registration / login
- Job search
- Job filtering
- Viewing job details
- Applying to a job

## Deliverables
- Test cases
- Bug reports
- UX findings
- Improvement proposals
- Final report

## Tools Used
- Manual testing
- Screenshot evidence
- Markdown documentation
- GitHub

## Repository Structure
- `test-cases/` → documented manual test cases
- `bug-reports/` → bug reports with severity and priority
- `ux-findings/` → usability issues identified
- `improvement-proposals/` → suggested product improvements
- `screenshots/` → supporting visual evidence

## Key Skills Demonstrated
- Manual testing
- Bug documentation
- UX analysis
- Critical thinking
- Product evaluation
