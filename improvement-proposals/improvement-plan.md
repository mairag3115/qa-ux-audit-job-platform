\# Improvement Proposals



\## Proposal 01 – Improve search guidance

Add helper text inside the search bar to clarify what users can search for.



\## Proposal 02 – Make filters more visible

Display active filters as removable tags so users can understand and control search results more easily.



\## Proposal 03 – Improve job card hierarchy

Highlight key information such as salary, location, and contract type for faster decision-making.



\## Proposal 04 – Add better application feedback

Show clear confirmation messages after each important action, especially after submitting an application.



\## Proposal 05 – Reduce cognitive load

Simplify pages with too much content and improve spacing, grouping, and visual prioritization.
